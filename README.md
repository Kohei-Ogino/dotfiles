dotfiles
====

my dotfiles


## Installation

```
$ git clone https://github.com/Kohei-Ogino/dotfiles.git
$ cd dotfiles/
$ sh dotfilesLink.sh
```

- vim-plug  
https://github.com/junegunn/vim-plug
```
$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

- Vim Plugin
```
:PlugInstall
```
